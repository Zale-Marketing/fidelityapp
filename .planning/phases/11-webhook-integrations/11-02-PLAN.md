Read the file to find where the card/card_holder insert completes successfully. After the card is inserted and before any redirect, add a fire-and-forget fetch call:

```typescript
// Fire-and-forget — dispatch webhook via server route (do NOT await)
const dispatchWebhook = (event: string, data: Record<string, unknown>) => {
  fetch('/api/webhooks/dispatch', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ merchantId: programData.merchant_id, event, data }),
  }).catch(console.error)
}

dispatchWebhook('nuovo_cliente', {
  card_holder_id: newCardHolder.id,
  full_name: formData.fullName,
  email: formData.email,
  program_id: programId,
})
dispatchWebhook('card_creata', {
  card_id: newCard.id,
  card_holder_id: newCardHolder.id,
  program_id: programId,
  merchant_id: programData.merchant_id,
})
```

Replace `programData.merchant_id`, `newCardHolder.id`, `newCard.id`, `formData.fullName`, `formData.email`, `programId` with the actual variable names used in the existing code. Read the file first to understand the exact variable names.
