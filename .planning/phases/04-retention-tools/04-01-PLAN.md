  <action>
Create `supabase/migrations/03_retention_schema.sql` with the following idempotent SQL:

1. Create `customer_tags` table (IF NOT EXISTS):
   - id UUID PRIMARY KEY DEFAULT gen_random_uuid()
   - merchant_id UUID NOT NULL REFERENCES merchants(id) ON DELETE CASCADE
   - name TEXT NOT NULL
   - color TEXT NOT NULL DEFAULT '#6366f1'
   - created_at TIMESTAMPTZ DEFAULT NOW()
   - UNIQUE(merchant_id, name)

2. Create `card_holder_tags` junction table (IF NOT EXISTS):
   - card_holder_id UUID NOT NULL REFERENCES card_holders(id) ON DELETE CASCADE
   - tag_id UUID NOT NULL REFERENCES customer_tags(id) ON DELETE CASCADE
   - PRIMARY KEY (card_holder_id, tag_id)

3. Add extended columns to `card_holders` (all ADD COLUMN IF NOT EXISTS):
   - contact_email TEXT
   - birth_date DATE
   - notes TEXT
   - marketing_consent BOOLEAN DEFAULT FALSE
   - acquisition_source TEXT
   - last_visit DATE
   - total_stamps INTEGER DEFAULT 0

4. Enable RLS on customer_tags and card_holder_tags with policies scoped via profiles subquery (matching the pattern in 01_notification_logs.sql):
   - customer_tags: "Merchants see own tags" USING (merchant_id IN (SELECT merchant_id FROM profiles WHERE id = auth.uid()))
   - card_holder_tags: "Merchants see own holder tags" USING (card_holder_id IN (SELECT id FROM card_holders WHERE merchant_id IN (SELECT merchant_id FROM profiles WHERE id = auth.uid())))
